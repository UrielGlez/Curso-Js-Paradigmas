.map() JavaScript

Built an Array like [1, 2, 3, ... , n] and the 'Calculate' button
will return a new array with the cube of each element

.filter() JavaScript

Built an Array like [1, 2, 3, ... , n] and the 'Calculate' button
will return a new array with the numbers greater than 2

.reduce() JavaScript

Built an Array like [1, 2, 3, ... , n] and the 'Calculate' button
will return the result of the subtraction of each element
